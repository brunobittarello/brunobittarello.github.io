Jump 'n Bump
Party game

Description

Jump ‘n Bump is a remake of a demo created in 1998 by Brainchild Design. Its code was released as public domain in 1999. I recreated the game logic and physics, without the original code, in Unity. I used the original sprites and audios.

The game’s theme is a deathmatch competition among rabbits. The player controls one rabbits and must step over other rabbit to kill it. The only controls are horizontal move and jump. There is no time up or max point, the players decides to end.

I improved this game adding more features. I created an IA to control a rabbit if the player set it. It allows single player game which wasn't possible in original version. Screens were added to configure match and game options, also splash screens were added.

Online and portability were implemented. Playing in online mode is possible and compatible with others platforms (ex: pc x android). Those feature wasn't present in original game too.

Production


Motivation

My cousins introduced me to this demo game when I was young. We played it a lot even with only one level available. That was fun. When I got knowledge enough to create games by myself, I decided to start doing a remake of this game. Jump ‘n Bump has a simplicity that I needed to begin make games. Some weeks after, I played my remake with my mother and my sisters and we had the same fun as I had playing the original in the pass.
