Cubo LudYx
Kids game

Description

This game is an auxiliary software. A toy called Cubo de Desenhar (Cube Drawing, in English) is a product of a LudYx store. This project was remunerated. Me and Natanael Rey (responsible to Game Design and Art).

The toy allows kids to draw on all cubes faces. Each faces has an animal borders drawn. With this game, the kids can import theirs draws by taking photos. For every face (will be) a unique game using a sprite cut from the photo put in the correspondent face. It allows them to play games with theirs draw.

Others minigames will be produce when the client contact us again.
