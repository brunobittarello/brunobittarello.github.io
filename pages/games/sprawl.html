Sprawl Celerity
Future race game

Description

Sprawl Celerity have a focus on speed, the idea is to make players feel they are driving a fast race car, and not a killing machine. But we want to provoke that nervous feeling on the players, especially the ones leading, since they are more propitious to being hit by weapons, this make the win feel much more rewarding. While weapons are an important part of the game, they are not required to succeed.

Motivation

My personal motivation is from an old game named Dethkarz. I've played a lot in my childhood and I never found something like that.

When I've discovered it would must create a large game project in the end of the game development course, I decided to create a game like Dethkarz.

Plauto accepted my invite to do this project with me. He is a programmer, so we two didn't have an artist to create models. Because of that, I got the art part and it was a good challenge for me.
