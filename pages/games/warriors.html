Strategy Warriors
Tactic artillery 2D game

Description

Strategy Warriors is a pseudo turn-based game in progress. It’s genre is artillery with a mechanics which requires each team sets movements to all it’s pieces. After all pieces of all teams are defined, all the movements are reproduced at the same time. When a piece gets hit, it stops to reproduce the recorded movements for it. The game ends when one team has no pieces left.

Every new update brings a new weapon. There is a lot of mechanics and screens to create in this project. Stay updated to the next features!

Motivation

Since I was young I like artillery games like Worms and golf games like Pangya!. However, in Worms I always thought that is strange when one team has one character and the enemy team has six and this one character can play more times than each character of the other team. In my game, every unit can play the same amount of seconds as any one.

Is easy to understand but is challenging find a way to make a correct and good solution for that issue. Collisions with teammates, collisions with enemies and destroyed pieces of terrain that used as a floor for some unit (makes a unit float) are examples of problems in this project. AI will be another hard challenge in this game.
