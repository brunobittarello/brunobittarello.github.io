Bomberman 3D
Party arcade game

Description

Bomberman is a very known franchise created by Hudson Soft. This game usually are a 2D game, with exceptions like Bomberman Ultra. But all of them have the third person gameplay.

After see this video, I wanted to create a game based on this ideia: bomberman in first person. Some features would be different and some other will appear in this different point of view.

Now this game is stoped, but wll be resumed in the future.
